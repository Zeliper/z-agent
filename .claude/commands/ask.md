# /ask Command

사용자가 질문을 할 때 사용하는 z-agent 명령어입니다.

## 사용법
```
/ask <질문>
```

## 예시
```
/ask 이 함수가 어떻게 동작하는지 설명해줘
/ask React 18에서 바뀐 점이 뭐야?
/ask 이 에러가 무슨 의미야?
```

## 처리 흐름

1. **난이도 분석** (Haiku)
   - 질문 복잡도 판정
   - H: 복잡한 개념, 아키텍처 관련
   - M: 코드 설명, 기술 개념
   - L: 간단한 조회, 검색

2. **Agent 선택**
   - H → Opus: 복잡한 설명, 깊은 분석
   - M → Sonnet: 일반적인 설명, 코드 분석
   - L → Haiku: 간단한 검색, 조회

3. **응답 생성**
   - 질문에 대한 답변 생성
   - 필요시 코드 예시 포함
   - 참고 자료 제공

## 실행 시 동작

이 명령어가 실행되면 다음을 수행하세요:

1. 질문의 복잡도를 분석합니다.
2. 적절한 Agent를 선택하여 답변을 생성합니다.
3. 간결하고 명확한 답변을 제공합니다.

## 특징

- Task 파일을 생성하지 않음 (단순 Q&A)
- 즉각적인 응답 제공
- 필요시 코드베이스 참조
- 복잡한 질문은 자동으로 상위 모델로 위임
